# HTMLCSS-PRICING-PLAN

HTML/CCS pricing plan page, capable of adapting to different screen resolutions.

# Demo Live

[**Click here for demo**]


# Target 

In mobile mode:

1. the plans stack vertically; as more space becomes available, they align side by side until reaching a 3-column layout in desktop mode, when space allows. ✔️ 

2. the order of the cards changes by bringing the Professional card (purple) to the first position, while keeping the natural order on desktop. ✔️ 
3. the Monthly/Yearly toggle becomes centered. ✔️ 

# Bonus

1. in “tablet” mode (i.e., when only 2 cards fit), the third card takes up the full width. ✔️ 
2. when a card takes up the full width, the feature list should be arranged in 2 columns. ✔️
